## 📖 Project Overview

Book Hive is a responsive online bookstore website developed using HTML5 and CSS3. The website provides users with an attractive and easy-to-navigate platform to explore books, learn about the store, create accounts, and manage their shopping experience.

The project was developed as a minor internship project to enhance front-end web development skills and gain practical experience in designing modern and responsive websites.

---

## ✨ Features

- Responsive and user-friendly design
- Attractive homepage with book showcases
- About Us page
- Books catalog page
- Shopping cart page
- User Login page
- User Registration (Sign Up) page
- Contact page
- Easy navigation between pages
- Modern UI using HTML and CSS

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Visual Studio Code
